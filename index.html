Data source

With a Google Maps API key, discovery and location context use Google's professional places database, and report maps can show an optimized driving route. Without a key, the tool falls back to free OpenStreetMap data, which can be thin in some regions.

How to get a free Google Maps API key (one-time, ~15 minutes)
  1. Go to console.cloud.google.com, sign in with any Google account, and create a new project.
  2. Google requires a billing account with a credit card for verification. Free monthly usage covers thousands of searches — at typical team volume the cost stays at $0.
  3. Enable five APIs: "Maps JavaScript API", "Places API (New)", "Geocoding API", "Routes API", and "Maps Static API". If the key has API restrictions, add all five there too.
  4. Go to Credentials, Create credentials, API key. Copy it.
  5. Strongly recommended: edit the key, and under "Application restrictions" choose "Websites", then add this tool's web address so the key only works on this page.
  6. Paste the key above and click Apply.

Step 03 · Location context

Read the map in plain language.

A factual orientation to the subject, competitive clusters, corridors, transit, and surrounding named places.

Location overview — per property

A factual read-out of the map is generated automatically: where the subject sits, how many listings and distinct mapped locations were found, distance bands, directions, measured clusters, corridors, and nearby named places. It describes what is visible without drawing conclusions about demand, rents, retention, or economic impact. Demographics, housing, and employment remain separate, cited source fields.

Step 04 · Facility evidence

Complete the competitor record.

Add rates, features, promotions, and source evidence for every included facility.

Guided competitor data collection

For each competitor: open its website, select the whole page (Ctrl+A), copy (Ctrl+C), paste into the box. The tool extracts rates and features automatically and records a citation for every value. Anything it cannot find is marked not available; it never guesses. Competitors are grouped by subject property.

✓
Privacy & data handling

Your research stays in this browser session.

This tool has no application database, user account, analytics pipeline, or research-storage backend. Your brief, selected competitors, collected evidence, and generated package remain in the active browser tab.

01 · PROCESSING

Browser-based processing

Parsing, calculations, duplicate consolidation, narrative assembly, report creation, and file exports happen directly in your browser.

02 · STORAGE

No research record is saved

The tool does not save briefs, addresses, competitor selections, pasted website evidence, reports, or exports. Closing or refreshing the tab clears the active research session.

03 · MAP SERVICES

Location requests go directly to providers

Addresses and map searches are sent from your browser to Google Maps Platform—or OpenStreetMap when used—to geocode locations, find places, draw maps, and calculate routes.

04 · API KEY

API key handling

No Google API key is embedded in this tool. A key you provide is held for the active session, and is stored in this browser only when you explicitly select “Remember this key”; no key is sent to an application backend.

Data flow

What happens to your information

1. You provide a briefIt is read and held in the current browser tab.
2. Providers return map dataOnly required location requests go to the named map services.
3. You export the resultFiles are generated locally and saved only when you choose to download them.
Data sources: Google Places via the Google Maps Platform when a key is supplied · OpenStreetMap contributors via Overpass API as fallback · Nominatim geocoding fallback · Statistics Canada, municipal planning pages, employment surveys and competitor public websites via user-guided collection with per-field citations.
This tool performs no automated scraping, stores nothing beyond an optional locally saved API key, and applies no analytical judgment. Verify all figures against cited sources before use in final materials.
