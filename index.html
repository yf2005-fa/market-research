Market research companion

COMPETITOR & LOCATION RESEARCH · V3

An independent research assistant for real estate due diligence. It receives a research brief covering one or many properties, locates competitors from professional map data, builds a location overview per property (geography, transit, demographics, housing pipeline, employment) from verifiable sources, and packages everything with full citations for internal analysis. It collects and organizes only — all judgment and synthesis stays with your internal tools.

🔒Nothing is stored. All processing happens locally in your browser tab. Data goes only to the map services used to locate competitors, and everything is erased when this tab closes.

Data source

With a Google Maps API key, discovery and location context use Google's professional places database. Without a key, the tool falls back to free OpenStreetMap data, which can be thin in some regions.

How to get a free Google Maps API key (one-time, ~15 minutes)
  1. Go to console.cloud.google.com, sign in with any Google account, and create a new project.
  2. Google requires a billing account with a credit card for verification. Free monthly usage covers thousands of searches — at typical team volume the cost stays at $0.
  3. Enable three APIs: "Maps JavaScript API", "Places API", and "Geocoding API".
  4. Go to Credentials, Create credentials, API key. Copy it.
  5. Strongly recommended: edit the key, and under "Application restrictions" choose "Websites", then add this tool's web address so the key only works on this page.
  6. Paste the key above and click Apply.

Location overview — per property

The market context behind the rents. Geographic context, transit access, and supply density are collected automatically from map data. Demographics, housing pipeline, and employment come from official public sources: the tool gives you the exact link, you copy the page, paste it here, and it extracts the numbers with a citation. Nothing is ever estimated — anything not found stays "Not available" for the analyst to see.

Guided competitor data collection

For each competitor: open its website, select the whole page (Ctrl+A), copy (Ctrl+C), paste into the box. The tool extracts rates and features automatically and records a citation for every value. Anything it cannot find is marked not available; it never guesses. Competitors are grouped by subject property.

Research package

One combined package covering every property: location overview narratives, competitor data, written summaries, full fact-check citations, verbatim source excerpts, and data gaps. Paste the text package into your internal AI, or open the formatted PDF report \u2014 it includes the map, per-property location overview, competitor sections with links, and the citation sheet and source excerpts as appendices. Use your browser\u2019s print dialog and choose "Save as PDF".

Re-check mode

Been a while since a package was generated? Paste it here. The tool re-runs discovery for the same subjects and radius, then reports what changed: competitors that disappeared, new entrants, and a verification checklist with direct links.

Data sources: Google Places via the Google Maps Platform when a key is supplied · OpenStreetMap contributors via Overpass API as fallback · Nominatim geocoding fallback · Statistics Canada, municipal planning pages, employment surveys and competitor public websites via user-guided collection with per-field citations.
This tool performs no automated scraping, stores nothing beyond an optional locally saved API key, and applies no analytical judgment. Verify all figures against cited sources before use in final materials.
